{
  "firstName":"Bob.",
  "lastName": "Marl"
}

An example JSON body when creating a team :

{
  "name":"DCU Ultimate",
  "shortHandName": "Blazers"
}

{
  "firstName":"Bob.",
  "lastName": "Marl",
  "playerNumber": 20
}
